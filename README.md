# ❤️ Heart Attack Possibility Prediction using Machine Learning (HAPP)

This project leverages machine learning classification algorithms to predict the likelihood of a heart attack based on patient clinical indicators and physiological data. Designed to aid early detection, the system processes healthcare metrics to evaluate and stratify patient cardiovascular risk.

---

## ✨ Key Features

* **Predictive Classification Engine:** Utilizes supervised machine learning models to analyze health parameters and evaluate heart attack susceptibility.
* **Feature Importance Analysis:** Highlights critical medical attributes (such as cholesterol levels, blood pressure, and age) that contribute most heavily to accurate risk tracking.
* **Data Preprocessing & Engineering:** Cleaned, scaled, and managed missing values or outliers within the health metrics datasets to eliminate model bias.

---

## 📂 Repository Structure

* **`HAPP/`**: Contains the core machine learning model scripts, dataset preprocessing pipelines, and exploratory data analysis (EDA) workflows.
* **`.gitignore`**: Excludes unnecessary local Python configurations, cache folders (`__pycache__`), and checkpoint files from tracking.

---

## 🛠️ Tech Stack

* **Language:** Python 3.x
* **ML Library Toolkit:** Scikit-Learn, Pandas, NumPy
* **Visualization Tools:** Matplotlib, Seaborn
